# Exercise 2: Protocol Exploration & REST APIs

## Objective
To deploy a local Node-RED middleware container and architect a RESTful HTTP server with GET/POST methods.

## Setup Instructions
1. Run the Node-RED Docker container.
2. Port mapping: `1880:1880`.
3. Endpoint URL: `http://localhost:1880/api/sensor`.

## Results
* **HTTP POST**: Successfully received JSON payload via Postman.
* **HTTP GET**: Successfully accessed via mobile browser on the same LAN.
